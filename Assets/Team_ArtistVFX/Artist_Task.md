# 美术与特效部门 (Artist & VFX) 开发任务与计划

欢迎来到美术与特效部门工作区。本目录将用于存放灵感参考、原画草稿或特效预制体的设计文档。正式资源请导入到项目的相关资源目录中（如 `美术素材` 文件夹）。

## 🎯 当前目标
为《倭瓜射手大冒险》提供生动、具有辨识度的视觉体验。在跑酷过程中通过动效和特效增加打击感和爽快感。

## 📋 任务列表

### 1. 2D 资源整理与动画制作 (Priority: High)
- **主角 (倭瓜射手)**：
  - 整理基础待机/行走的 Sprite。
  - 制作攻击动作（嘴部发射豌豆）和跳跃/落地时的形变动画（可以结合程序的缓动插件制作果冻效果）。
- **敌人**：
  - 导入并切分敌人图片。
  - 为敌人制作简单的 idle 呼吸动画，以及受击时的闪白/碎裂帧。
- **场景与 UI**：
  - 绘制并切割无限滚动场景图层，包含：**草地层（前景跑道）**、**泥土层（地基）**和**背景层（远景）**。
  - **重要**：所有场景图层必须支持首尾相连的无缝拼接（Seamless），保证双图交替滚动时无断层。
  - 设计符合游戏搞怪可爱风格的 UI 界面（分数面板、结算面板和开始按钮）。

### 2. 视觉特效 (VFX) 设计 (Priority: Medium)
使用 Unity 粒子系统 (Particle System) 或 Shader Graph 制作以下核心反馈特效：
- **射击反馈**：
  - 枪口/嘴部开火特效 (Muzzle Flash)。
  - 子弹飞行拖尾 (Trail) 效果。
- **命中与死亡反馈**：
  - 敌人被击中时的汁液飞溅或粒子爆开特效。
- **环境互动**：
  - 玩家落地时脚底产生的扬尘特效。
- **特殊武器表现**：
  - **玉米大炮**：夸张的大范围爆炸火光特效和烟雾。
  - **豌豆机枪**：高频射击时的幻影或火花堆叠效果。

### 3. 插件协同 (Priority: Medium)
- 与程序部门合作，将准备好的特效资源与 `Feel` 等插件结合，打磨“屏幕震动”、“命中顿帧 (Hit Stop)”等高级表现手法。

## 🗓️ 交付与反馈
- 优先提供一批占位/基础切图资源（角色的基础形态与背景）供程序搭建测试环境。
- 逐步迭代 UI 和高级 VFX 粒子效果。
