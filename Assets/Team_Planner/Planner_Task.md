# 策划部门 (Planner) 开发任务与计划

欢迎来到策划部门工作区。本目录将用于存放所有的数值配置表、关卡设计文档和系统设计细节。

## 🎯 当前目标
为《倭瓜射手大冒险》提供坚实的数值支撑和节奏体验，确保游戏从易到难的平滑过渡，并在后期带来爽快的挑战感。

## 📋 任务列表

### 1. 数值表与难度曲线设计
跑酷游戏的灵魂在于节奏控制。请着手设计以下数值：
- **背景与地面滚动速度**：初始速度设定，以及随时间（或分数）递增的计算公式/阶梯。
- **敌人生成规律**：
  - 最小生成间隔时间与最大生成间隔时间。
  - 生成频率随时间增加的变化曲线。
  - 单次生成数量（1个或多个）的概率分布。

### 2. 武器与道具配置
为高空中的特殊武器配置详细参数：
- **玉米大炮**：
  - 拾取概率 / 出现条件。
  - 炮弹射击间隔时间。
  - 爆炸伤害范围（以 Unity 计量单位为准的半径范围估算）。
  - 持续时间（当前暂定10秒，可视测试情况调整）。
- **豌豆机枪**：
  - 拾取概率 / 出现条件。
  - 极速射击的频率（如 0.1秒/发）。
  - 持续时间（当前暂定10秒）。

### 3. 分数与结算机制
- 确定每个敌人的基础击杀分数。
- 确定连续击杀或特殊击杀是否需要引入连击(Combo)分数加成。

## 🗓️ 交付与反馈
- 请在设计出第一版数值后，与程序部门对接，将数值填入对应的 `ScriptableObject` 或 `GameManager` 的公开面板中。
- 组织游戏早期的灰盒测试，根据团队反馈不断微调“难度曲线”。
