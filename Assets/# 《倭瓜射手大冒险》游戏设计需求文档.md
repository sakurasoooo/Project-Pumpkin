# 游戏设计需求文档

## 🎯 游戏概述

**核心玩法**：横版自动前进的弹跳射击跑酷游戏
**主角设定**：融合倭瓜身体与豌豆射手特征的混合植物，通过射击与跳跃消灭敌人

## 👾 主角设定

### 倭瓜射手特征

- **外观**：圆滚的倭瓜身躯 + 豌豆射手的标志性发射嘴
- **基础移动**：持续小幅度蹦跳前进
- **自动攻击**：自动从嘴部发射豌豆子弹攻击前方敌人

## 🎮 核心游戏机制

### 1. 自动射击系统

- 倭瓜自动向前方发射豌豆子弹
- 子弹命中敌人即可消灭目标
- 射击频率保持稳定节奏

### 2. 跳跃攻击系统

- **点击触发**：玩家点击屏幕时倭瓜执行跳跃，可以二连跳

### 3. 敌人生成规则

- **生成方式**：在前进路径上间断性生成，可能生成一个，可能生成多个
- **敌人特性**：保持idle状态，作为射击和跳跃的目标障碍，可以自行添加一些效果使其看起来生动

## 🏰 场景与道具系统

### 武器道具

在场景高处设置特殊武器道具：

| 武器类型           | 攻击方式           | 持续时间 | 战略价值     |
| ------------------ | ------------------ | -------- | ------------ |
| **玉米大炮** | 发射大范围爆炸炮弹 | 10s      | 大范围伤害   |
| **豌豆机枪** | 高速连续发射豌豆   | 10s      | 持续火力压制 |

### 动态难度系统

- **速度递增**：游戏背景滚动速度随时间逐渐加快
- **节奏变化**：敌人生成频率随进度提升
- **难度曲线**：为玩家提供从入门到精通的无缝体验

## 📊 计分与胜利条件

### 得分系统

- **基础击杀**：每消灭一个敌人获得固定分数
- 游戏界面：游戏中实时展示更新玩家分数

### 游戏结束条件

- **失败条件**：倭瓜接触到敌人则失败
- **胜利条件**：无明确终点，以获得高分为目标
- **持续挑战**：地图无限循环，速度持续增加，考验玩家极限

## ✨ 设计要求

1. 尽量优化效果（效果加分占比较高)，使用缓动等方式使人物和敌人表现的更动态一些，敌人要有受击反馈，ui要让人一目了然
2. 文件夹中有相关美术素材，也可自行添加新的美术素材使其画面表现更棒
3. 可自行优化游戏逻辑使其更好玩
4. 操作要让玩家感觉连贯顺畅
5. 打包成apk并测试没有卡死bug（如无法打包apk可以联系HR沟通)
